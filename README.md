# Eco Board 
Веб-приложение для размещения объявлений об экологических акциях, волонтёрстве и эко-подработках.

##  Функционал
- Добавление объявлений
- Поиск объявлений
- Просмотр объявлений


##  Технологии
- Java
- JDBC
- MySQL
- JSON (Gson / org.json)


##  Архитектура
Клиент-серверное приложение  
Сервер взаимодействует с БД через JDBC  
Данные передаются в формате JSON


##  Запуск проекта
1. Настроить базу данных MySQL
2. Указать параметры подключения в конфиге
3. Запустить сервер
